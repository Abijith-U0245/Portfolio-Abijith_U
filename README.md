# Abijith U — Portfolio

Personal portfolio website showcasing my projects, experience, achievements, and journey as a Software Engineer & AI/IoT builder.

🔗 **Live Site:** [abijithu.netlify.app](https://abijithu.netlify.app/)

## About

This portfolio highlights my work across full-stack development, AI/ML, IoT hardware, and competitive programming, including:

- **GigEase** — AI-powered parametric insurance platform (Top 10, Guidewire DevTrails 2026)
- **ShruthiAI** — Offline Hindi voice assistant on Raspberry Pi (ARM Bharat AI SoC Finalist)
- **CARES** — RAG-based AI document intelligence platform
- **Smart Fingerprint Attendance System** — ESP32-based biometric IoT device
- **ARISE** — Academic ERP & student risk analysis platform
- **STEM4ALL** — Collaborative STEM education platform

It also documents my journey as **Secretary of GDG On Campus, Chennai Institute of Technology**, hackathon finals, workshops conducted, and volunteering at CIT's Embedded Systems & IoT Centre of Excellence.

## Tech Stack

- **HTML5, CSS3, JavaScript (Vanilla)** — single-page, no build step
- **Google Fonts** — Instrument Serif, Satoshi, JetBrains Mono
- Fully responsive, animated on scroll, with an interactive lightbox gallery

## Sections

- Hero with animated role typewriter

## Run Locally

Just open `index.html` in any browser — no dependencies, no build process required.

```bash
git clone https://github.com/Abijith-U0245/Portfolio-Abijith_U.git
cd Portfolio-Abijith_U
open index.html
```
